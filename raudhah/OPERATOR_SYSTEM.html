Login & Redirect

Admin (Operator)

  • URL Login: http://localhost/raudhah/login.php
  • Username: admin
  • Password: Admin@1234
  • Redirect to: /admin/dashboard.php

User (Member)

  • URL Login: http://localhost/raudhah/login.php
  • Username: [member username]
  • Password: [member password]
  • Redirect to: /dashboard/index.php

Fitur Admin Operator

1. Kelola Member

  • Lihat daftar semua member
  • Aktifkan/nonaktifkan akun member
  • Edit data member (nama, email, nomor rekening)
  • Lihat riwayat aktivitas member

2. Verifikasi Pembayaran

  • Lihat daftar pembayaran yang pending
  • Verifikasi bukti transfer
  • Tolak pembayaran dengan alasan
  • Akses laporan pembayaran

3. Kelola Pohon Binary

  • Lihat struktur pohon member
  • Edit posisi member (kiri/kanan)
  • Edit parent/referrer member
  • Lihat visualisasi pohon

4. Kelola Paket Umroh

  • Tambah paket baru
  • Edit paket yang ada
  • Hapus paket
  • Manage quota dan keberangkatan

5. Laporan Komisi

  • Lihat komisi pending, approved, paid
  • Verifikasi komisi
  • Kelola penarikan komisi (withdrawal)

Database Schema

users table

role ENUM('admin', 'user')
- admin = Operator yang mengelola sistem
- user = Member/Jamaah yang terdaftar

status ENUM('pending', 'active', 'suspended')
- pending = belum verifikasi
- active = aktif dan bisa mengakses
- suspended = dinonaktifkan operator

registrations table

payment_status ENUM('unpaid', 'pending_verification', 'verified', 'rejected')
- Dikelola oleh operator (admin)

withdrawals table

status ENUM('pending', 'approved', 'rejected', 'completed')
- Operator mengelola approval dan pembayaran

Access Control

Untuk melindungi admin panel:

// Hanya admin yang bisa akses
requireAdmin(); 
// atau 
isAdmin(); 

// Atau gunakan alias
isOperator();
